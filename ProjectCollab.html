About the Project

Tabster enables sit down restaurants to accept preorders for food. The customer chooses the restaurant, selects their food within the app when they arrive, and waits to be seated. By the time the customer sits down, they will have their first course and drinks in minutes. This increases table turnover for the restaurant and leaves the customer with a good dining experience. As an added bonus, customers will be prompted by notification upon leaving the restaurant to rate the menu items they ordered and their experience at the restaurant.

GitHub is a web-based Git repository hosting service, which offers all of the distributed revision control and source code management (SCM) functionality of Git as well as adding its own features. Unlike Git, which is strictly a command-line tool, GitHub provides a web-based graphical interface and desktop as well as mobile integration. Dropbox is a file hosting service operated by Dropbox, Inc., headquartered in San Francisco, California, that offers cloud storage, file synchronization, personal cloud, and client software. Dropbox allows users to create a special folder on their computers, which Dropbox then synchronizes so that it appears to be the same folder (with the same contents) regardless of which computer is used to view it. Files placed in this folder are also accessible via the Dropbox website and mobile apps. Dropbox uses a freemium business model, where users are offered a free account with a set storage size and paid subscriptions for accounts with more capacity.[6]
